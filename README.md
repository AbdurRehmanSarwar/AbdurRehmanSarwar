### 👋 About

Senior Data &amp; AI Engineer with **7+ years** designing and scaling production data platforms
across **AWS, Azure, and GCP**. I build high-volume **ETL/ELT**, batch, and streaming pipelines
(**Airflow, dbt, Kafka, Spark**) feeding modern warehouses (**Snowflake, Redshift, BigQuery**),
and I ship **LLM applications** — RAG, multi-agent systems, and CAG/KAG architectures. I care about
cutting data latency, improving pipeline reliability, and turning raw data into products that drive
real decisions.

- 🔭 **Now:** Senior Software Engineer (Data &amp; AI) — production pipelines, RAG analytics apps, and autonomous multi-agent systems.
- 🌱 **Exploring:** agent orchestration (LangGraph), LLM evaluation, and cost/latency optimisation for inference.
- 🤝 **Open to:** senior Data / AI Engineering roles (remote, flexible time zones).
